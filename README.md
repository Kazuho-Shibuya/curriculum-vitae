## 職務経歴
### [株式会社TENT（2021/09-）](https://tent-inc.jp/)

#### 職務要約
- クラウド型のレンタルモール[カウリル](https://kauriru.com)の設計、開発、テスト
- カウリル出店企業向けの管理画面の設計、開発、テスト

#### 職務内容
##### 使用技術
Ruby on Rails、React、Docker、AWS、Github

##### 実績
- CTO、業務委託の方を含めたエンジニア3人のみで、カウリル本体とカウリル管理画面を開発
- 個人では主にRailsを使い管理画面を担当し、月間平均10issueを消化

##### 課題解決
###### 課題
- カウリルでは企業が出店するのにあたり、必要な情報（ストア紹介、特定商法取引に基づく表記、レンタル規約、プライバシーポリシー、運営会社情報、キャンセルポリシー、破損等について）をスプレッドシート上で出店企業とメールでやり取りしていた。
- 1企業あたり記入していただくのに2週間＋いただいた内容の確認で1日＋修正点の記入が1週間で合計1ヶ月ほどかかり、出店が決まってから実際に出店するまで少なくとも1ヶ月ほどかかってしまっていた。

###### 解決
- スプレッドシート＋メールではなく、管理画面から上記必要情報を入力させてカウリルに自動反映するようにした。
- サンプル文のチェックボックスを用意し、簡単に必要事項の記入ができるようにした。
- この対応により、必要情報は早ければ当日に企業が入力完了し、出店準備が完了するようになった。
- よって、当日入力しない企業、情報入力に時間がかかる企業などを考えても、少なくとも1企業あたり2週間（80時間分）は出店準備にかかる期間を短縮することができた。
